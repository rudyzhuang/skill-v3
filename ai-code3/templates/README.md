# templates（authoring 指针）

`stages.json`、`config.dev.json` / `config.release.json` 等 **JSON 字段形状** 的真源在本仓库：

- `docs/templates/stages.json.template`
- `docs/templates/config.dev.json.template`
- `docs/templates/config.release.json.template`

发布 skill 时若需随包分发模板，须与上述目录**同版本**拷贝或经构建脚本同步（见 `docs/spec/code3.md` §17）。
